# Student-Database-Management-System.

A Java-based CRUD application that allows users to manage student records — including adding, updating, deleting, and viewing students — using a MySQL database.

>Features

  -Add, update, delete, and view student details

  -Stores data securely in a MySQL database

  -Command-line interface for easy interaction

  -Handles database connectivity using JDBC

>Technologies Used

  -Programming Language: Java (Core Java, JDBC)

  -Database: MySQL

  -Connector: MySQL Connector/J (JDBC Driver)
